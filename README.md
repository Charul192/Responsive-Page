# 🐾 AdoptMe - Pet Adoption Website

A simple and responsive website for adopting pets, built using HTML and CSS. It features beautifully styled pet cards with images and descriptions, encouraging users to give animals a loving home.

---

## 🌟 Features

- Responsive card layout for pets
- Clean and minimal UI
- Mobile-friendly design
- Styled using pure CSS
